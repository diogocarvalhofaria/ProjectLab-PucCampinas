# ProjectLab - Gestão de Laboratórios PUC-Campinas

Este projeto é uma solução Full Stack para o gerenciamento de laboratórios acadêmicos, desenvolvido como parte de um teste técnico.

## Tecnologias Utilizadas

- **Backend:** .NET 10, Entity Framework Core, MySQL.
- **Frontend:** Angular 19, Reactive Forms, TypeScript.
- **Infraestrutura:** Docker & Docker Compose.
- **Documentação:** Swagger/OpenAPI.

## Como Rodar o Projeto

### 1. Backend (Docker)

Certifique-se de ter o Docker Desktop instalado e rodando. Na pasta raiz, execute:

```bash
docker-compose up --build
```
